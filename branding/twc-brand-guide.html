Who we are

Brand Story

The Wall Chart Co makes premium digital downloads for football and rugby fans who want to get more out of the tournament. Every product opens in any web browser — no app, no subscription, no install. Buy once, use all tournament long.

Founded in Swansea, Wales by Anthony — 30 years of web development experience combined with a genuine love for football and rugby. We build the kind of tools we would want to use ourselves watching the matches.

Brand Values

  • Quality over quantity — Every product is meticulously designed with correct data and clean layouts
  • Simplicity — One HTML file, any browser, no install. It just works
  • Fan-first — Built by fans, for fans. We care about the experience of following the tournament
  • Honest pricing — Premium quality at fair digital prices. No subscriptions, no upsells
  • Welsh pride — Made in Swansea, Wales. Local craft, global audience

Brand Voice

Confident, knowledgeable, approachable. We know our stuff but we don't talk down to anyone. We write like a mate who happens to be really good at building web tools and knows a lot about football and rugby.

Colour Usage Rules

  • Navy — Primary background for headers, sidebars, section heads. The anchor of the brand
  • Gold — Accent colour for logos, borders, highlights, CTAs, active states. The premium signal
  • Red — Secondary accent for dates, times, sport-specific highlights, urgency
  • Cream — Content background, alternating table rows, print-friendly surfaces
  • White — Input fields, score boxes, clean content areas

Sport-Specific Accent Colours

  • Football products — Use core palette (navy/red/gold). Team editions use team colours as accent override
  • Rugby products — Core palette with green (#2D6A2E) as supplementary accent for rugby-specific elements
  • Champions League — Core palette with UCL blue (#1A3668) integration
  • Premier League — Core palette with PL purple (#38003C) as theme option

Typography Rules

  • EB Garamond: Brand name, product titles, group letters, section headings, trophy/champion text
  • Inter: Navigation, body text, data tables, buttons, labels, UI elements, score inputs
  • Section labels: 8-10px, Inter, weight 700, letter-spacing 0.15-0.2em, uppercase, gold colour
  • Never mix — Garamond is the premium voice, Inter is the functional workhorse

Logo Rules

  • Monogram is always "TWC" in EB Garamond Bold inside a rounded square
  • Default: Gold monogram on navy background, white wordmark
  • On light backgrounds: Navy monogram with gold text, navy wordmark
  • Minimum monogram size: 26px (toolbar), standard: 44px (header), large: 64px (brand page)
  • Always include the tri-colour stripe (red/white/navy) above or below the header
  • Gold (#C8A94A) border always separates the header from content below
Product Components
Reusable design patterns across all products

Standard Product Structure

  • Tri-colour stripe — 4-5px gradient bar at very top of every product
  • Sticky header — Navy background, TWC logo left, product title centre, action buttons right
  • Gold border — 2-3px gold line below header, above footer
  • Tab navigation — Uppercase labels, gold active state, navy/dark background
  • Content cards — Rounded corners (8-10px), subtle border, dark background on dark theme
  • Score inputs — White background, 1px border, centered text, rounded corners
  • Print button — Gold background, navy text, uppercase, always visible
  • Footer — Navy background, gold TWC brand, copyright, unofficial disclaimer

Interactive Features (Standard across all products)

  • localStorage auto-save on every score entry
  • Export save as JSON file with timestamped filename
  • Import save from JSON file with validation
  • Print/Save PDF button with @media print optimisation
  • Theme variants (minimum 3 per product)
  • Responsive layout — works on desktop, tablet, and phone
  • Flag images from flagcdn.com (free, no API key needed)
  • All times shown in UK GMT/BST with clear labelling

Etsy Listing Standards
Consistency across all listings

Title Format

[Tournament] [Year] [Product Type] | [Key Feature] | Digital Download

Example: "World Cup 2026 Interactive Wall Chart | Track All 104 Matches | Digital Download"

Keep under 140 characters. Front-load the tournament name. Always end with "Digital Download".

Description Structure

  • Opening line: One clear sentence about what the product does
  • "Open the file in any browser" line
  • WHAT YOU GET section with bullet points
  • HOW IT WORKS section (download → open → use)
  • UNOFFICIAL NOTICE disclaimer
  • DELIVERY section (instant digital download)
  • ALSO AVAILABLE cross-sell section
  • Copyright notice

Tags (13 per listing)

  • Tags 1-3: Tournament + product type (highest intent)
  • Tags 4-6: Sport + year + variation keywords
  • Tags 7-9: Gift/occasion keywords
  • Tags 10-13: Broader sport/download keywords for discovery
  • Always include: "digital download", tournament year, sport name

Photography / Mockups

  • Image 1: Product screenshot on laptop mockup, 45-degree angle, lifestyle setting
  • Image 2: Full product screenshot (browser view)
  • Image 3: Close-up of key feature (group table, bracket, team page)
  • Image 4: Mobile/tablet view showing responsive design
  • Image 5: "What's included" infographic with TWC branding
  • Consistent gold/navy colour framing on all mockup images
