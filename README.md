## How It Works

1. Customer buys on Etsy (instant digital download)
2. Opens the HTML file in any browser (Chrome, Firefox, Safari, Edge)
3. Enters scores as matches are played
4. Tables and brackets update automatically
5. Progress auto-saves in the browser
