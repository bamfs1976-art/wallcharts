All videos should be screen recordings of the actual products in a browser window, filmed with your phone directly off a laptop/monitor screen, or exported as screen recordings. 9:16 crop works well showing the product UI top-to-bottom with overlaid text. Use the product card mockups as thumbnail images.

Recording Setup

  • Use Windows + G (Xbox Game Bar) for free screen recording
  • Or OBS Studio (free) set to 1080×1920 portrait crop
  • Record in a dark room — the navy UI looks great on camera
  • Zoom browser to 125% for better readability on TikTok
  • Record at 60fps if possible for smooth scroll/input animations

Editing in TikTok App

  • Trim to the right length in-app (no editor needed)
  • Add text overlays using TikTok's built-in text tool
  • Use Auto Captions for accessibility and reach
  • Add trending sound AFTER recording — music boosts reach
  • Post on Tuesday–Thursday 7–9pm for max engagement
